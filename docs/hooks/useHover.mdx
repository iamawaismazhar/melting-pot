# useHover

```
const { hover, bind } = useHover();
```

## Examples

<Playground>
  {() => {
    const { hover, bind } = useHover();
    const styles = {
      wrapper: {
        height: 300,
        display: 'flex',
        flexDirection: 'column',
        alignItems: 'center',
        justifyContent: 'center',
        cursor: 'pointer',
        backgroundColor: hover ? 'yellow' : 'white',
        color: 'black',
      },
      emphasis: {
        color: 'tomato',
        textTransform: 'capitalize',
      },
    };
    return (
      <div {...bind} style={styles.wrapper}>
        <p>Hover on the white background to change the hover status</p>
        <p>
          Status: <span style={styles.emphasis}>{String(hover)}</span>
        </p>
      </div>
    );
  }}
</Playground>

## API

useHover() returns a `boolean` `'hover'`, which indicates if specified container hovered or not.

<PropsTable
  properties={[

    },
    {
      name: 'bind',
      type: 'o',
      description: 'returns `onMouseEnter` & `onMouseLeave` which we can spread to the element',

  ]}
